HTML

Es el lenguaje estándar utilizado para crear páginas web. Es un lenguaje de marcado que define la estructura y el contenido de una página web mediante etiquetas y elementos.
DOCTYPE
Es la etiqueta que define que el documento es HTML.
HTML
Este elemento envuelve todo el contenido de la página.
HEAD
Esta sección contiene metadatos sobre el documento, No se muestra en la ventana del navegador.
BODY
Esta sección contiene todo el contenido visible de la página web, como texto, imágenes, enlaces, formularios, etc.

Etiquetas HTML Más Usadas

Etiqueta Descripción
<p> Esta etiqueta se utiliza para representar un párrafo de texto en una página web.
<h1> - <h6> Estas etiquetas se utilizan para definir encabezados, con <h1> siendo el más importante y <h6> el menos importante.
<a> Esta etiqueta crea un enlace a otra página web o a un recurso dentro de la misma página.
<img> Se utiliza para insertar imágenes en una página web.
<ul>, <ol>, <li> Estas etiquetas se utilizan para crear listas no ordenadas (<ul>), ordenadas (<ol>), y elementos de lista (<li>).
<input> Esta etiqueta se utiliza para crear elementos interactivos dentro de formularios como campos de texto, botones, etc.
<form> Se utiliza para crear un formulario que recoge datos introducidos por el usuario.
<table>, <tr>, <td> Estas etiquetas se utilizan para crear tablas, filas y celdas en una página web.

CSS

Es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita. Funciona junto con el lenguaje HTML que se encarga del contenido básico de los sitios.
Es utilizable en todos los navegadores y plataformas Debido a que es un lenguaje popular para dar formato a los sitios web, su uso es universal para un amplio número de dispositivos,
Optimiza el funcionamiento de las páginas web. Al separar el código de contenido y de estilo, es mucho más rápido el procesamiento de la información, lo cual se traduce en una experiencia más fluida para los usuarios y en una carga de trabajo menor para los procesadores.
Permite personalizar totalmente la apariencia de las páginas. El uso de diferentes códigos de colores y de fuentes permite emplear paletas de muchísimas tonalidades y múltiples tipografías.

JavaScript

es un lenguaje de programación diseñado en un principio para añadir interactividad a las páginas webs y crear aplicaciones web.

Construcción de servidores web y aplicaciones de servidor.
Desarrollo de aplicaciones para dispositivos móviles, híbridas o que compilan a nativo.
Desarrollo de aplicaciones de escritorio para sistemas Windows, Linux y Mac.
Desarrollo de juegos

SCRUM

es un marco de trabajo ágil utilizado principalmente en el desarrollo de software, aunque sus principios y prácticas pueden aplicarse a otros tipos de proyectos. Es parte de un enfoque más amplio conocido como Agile, que se centra en la flexibilidad, la colaboración y la entrega continua de valor.
